快速上手
5 分钟跑起来

本地运行，不需要账号，不上传你的数据。

1

下载项目

从 GitHub 下载或 clone 到本地。

git clone https://github.com/atom-xu/VilCc.git
2

安装依赖

一条命令装好所有需要的 Python 包。

pip install -r requirements.txt
3

启动服务

启动后会自动在本地开一个管理界面。

uvicorn main:app --port 8765
4

打开管理界面，一键配置 AI 客户端

访问 localhost:8765，在「Connect to AI」区域点击对应客户端的「一键连接」按钮，配置完重启 AI 客户端即可。

5

直接让 AI 去抓内容

回到 Claude 或 Cursor，粘贴视频链接告诉 AI 你想干什么，它会自动调用 VilCC。
